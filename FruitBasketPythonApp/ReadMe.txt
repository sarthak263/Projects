To run python file. 
	1. Please be in the directory FruitBasketPythonApp
	2. on Command Line run:  python3 Main.py {$PATH}. If no path is given. It will print out a message saying "File was not found. Exiting.."
	3. If the file is not CSV structure it will print "Cannot read the given file. Exiting.."


	Please use basket.csv to test the code or any other readable csv file.
