To Run:
	javac MandeBrot.java
	java MandeBrot
	
for Arguments:
	if none given. It will run Snowman, Field, Dragon and my created Image named "Coral"
	
	if 7 arguments given without the name of MandeBrot. it will create your image with the inputs provided but
	it will simply name the save jpg file to "yourCreation.jpg". If, in any case you have not provided the correct inputs
	such as not providing -NUMTHREADS, -XHI, -XLO, -YLO, -YHI, -THRESHOLD, -SIZE with its correct corresponding values it will simply provide you with 
	a message saying please enter correct inputs. However the order does not matter, just need to have correct format.
	
	
	ex. to run with arguments: -NUMTHREADS {values} -XLO {values} -XHI {values} -YLO {values} -YHI {values} -THRESHOLD {values} -SIZE {values} -NAME {values}
	
	for my own created image named Coral. 
	I used the inputs -XLO 0.3469233141 -XHI 0.3481312224 -YLO -0.368 -YHI 0.365 -THRESHOLD 200 -SIZE 2000
